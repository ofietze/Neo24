# Neo24

24 hour clock that highlights the amount of day light.
Check it out [here](https://ofietze.github.io/Neo24/).

### _Features_:

- Displays current Time in 24 hour format
- Neomorphic Interface
- Added Themes: Dark Mode <-> Light Mode
- Using Sass
