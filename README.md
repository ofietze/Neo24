
# Neo24


### *Features*:
* Displays current Time in 24 hour format
* Neomorphic Interface 
* Added Themes: Dark Mode <-> Light Mode


## *Features to be added*

- Adding markers for sunrise and sunset 
- Adding hour labels for easier reading


## *Technologies used*

- HTML5
- CSS3 (with Flexbox)
- Vanilla Javascript
